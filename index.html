Features

Group Management

This feature allows group admins to manage members, set rules, and monitor group activities. Easily add or remove members, assign roles, and set permissions.

Menfess

This feature allows users to send anonymous confessions through the bot. Messages are sent without revealing the sender's identity, ensuring privacy and confidentiality.

Media Downloader

This feature allows users to download media from various platforms such as YouTube, Spotify, SoundCloud, TikTok, Instagram, Twitter, and CapCut. Easily download videos, audio, or other content.

Converter

This feature allows users to convert various file types to other formats. For example, convert videos to audio, images to PDF, and more. The conversion process is quick and efficient.
